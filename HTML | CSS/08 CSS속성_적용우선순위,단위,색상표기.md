# CSS 필수 속성 다루기
## 기본 스타일 시트
웹 브라우저에 기본으로 내장되어 표시되는 요소들의 글꼴이나 크기, 색상 굵기 등 미리 정의된 파일(문서)양식.  
웹 브라우저마다 전부 독자적으로 스타일이 정의된 기본 스타일 시트가 있다. (브라우저마다 차이가 있는 이유)  
이미 스타일 적용된 상태로 표시가 되는 태그들이 있다. (`<hn>` 태그 등 ...)  
### 적용 우선순위
> 사용자 정의 스타일 시트 > 기본 스타일 시트


CSS는 단계적으로 스타일이 적용되기 때문에 중복 되었을 경우, 마지막에 영향을 주는 하나의 스타일만 적용이 된다.  
> **CSS의 개별성 규칙 (점수)**  
> 전체 선택자 (`*`) : 0점  
> 태그 선택자 : 1점  
> 가상 요소 선택자 (`::before`, `::after` ...) : 1점  
> 클래스 선택자 (`.`) : 10점  
> 가상 클래스 선택자 (`:hover`, `:visited`, `:link` ...) : 10점  
> 아이디 선택자 (`#`) : 100점  
> 인라인 스타일 (`style=""`) : 1000점  

사용된 선택자 종류를 파악, 점수를 계산하는 방식이다.  
우선순위로 보자면 `!important > 인라인 스타일 > 아이디 > 클래스(가상클래스 포함) > 태그(가상요소 포함) > 전체 선택자` 순이다.  
`!important`는 다른 선택자들을 압도, 우선 적용이 된다.  
선택자를 최대한 자세히 적을 수록 점수가 높다고 생각하면 된다.  
### 상속 inherit
부모 요소에 적용된 스타일을 자식 요소가 물려받아 적용되는 현상.  
상속 되는 속성이 있고, 상속이 안되는 속성이 있다.  
```html
<head>
    <style>
        div {
            color: red;
        }
    </style>
</head>
<body>
    <div>
    	<p>여기서 div로 지정된 스타일이 적용이 된다.</p>
    </div>
</body>
```
CSS는 div에만 지정되어있지만 `<p>` 태그가 물려 받아 폰트 색상이 적용이 된다.  
### 단위
**절대 단위** : 어떤 환경이라도 동일한 크기로 보인다. (px)  
**상대 단위** : 부모 요소, 웹 브라우저의 창 크기에 따라 상대적으로 결정되는 단위.  
#### 상대 단위
% : 해당 속성의 상위 요소 속성 값에 대한 상대적인 크기  
em : 부모 요소의 텍스트 크기 기준  
rem : html 태그의 텍스트 크기 기준  
vw : 뷰포트 너비 기준  
vh : 뷰포트 높이 기준  

### 색상 표기법
#### 키워드 표기법
```css
p {
	color: blueviolet;
}
```
색상을 영문명으로 지정하는 방법으로 실무에서는 잘 사용하는 편이 아니다.  
#### RGB 색상 표기법 
```css
h1 {
	color: rgb(225, 0, 0); /* Red, Green, Blue 값을 순서대로 넣어준다. */
}

h2 {
	color: rgba(0, 255, 0, .5); /* 알파 값은 0부터 1 사이의 소수로 표기. 이때 0은 생략 가능 */
}
```
빨간색, 초록색, 파란색을 이용해 특정 색을 표현하는 방법.  
추가로 색의 투명도를 지정(alpha 값) 할 수 있다.  
#### HEX 표기법
```css
p {
    color: #939393;
}
```
Red, Green, Blue에 해당하는 값을 16진수로 변환, 00~ff로 나타낸다.  
실무에서 가장 많이 사용하는 색상 표기법.  
> #RRGGBB

앞에 `#`을 붙여 표기, '00'값이 가장 낮은 값, 'ff'값이 가장 높은 값이다. (00 < ff)
