CSS 적용하는 방법

html 문서 안에서 CSS 적용하는 방법은 두가지가 있다.
하나는 인라인 스타일로 html 태그에 직접적으로 style 속성으로 작성하는 방법이고,
또 다른 하나는 내부 스타일 시트를 사용하는 방법으로
주로 head단에 style 태그로 CSS 작성한다.

또 다른 방법으로는 따로 외부에 CSS 파일을 만들어 사용하는 방법이 있다.

CSS 선택자

기본 선택자 사용하기

전체 선택자

모든 요소들을 모두 한번에 지정하는 방법이다.
body 태그 뿐만이 아닌 html, head 태그들을 비롯한 html 문서에 있는 모든 태그들을 말한다.

태그 선택자

특정 태그를 선택자로 지정하게 되면, 지정한 태그들의 모든 요소가 선택된다.

아이디 선택자

id 속성값으로 선택자를 지정하는 방법이다.
id는 하나의 HTML문서에 고유한 값, 단독으로만 쓰여야 한다.
한번 쓰였으면 해당 문서에서는 다시 지정할 수 없다.

클래스 선택자

실무에서 가장 많이 사용하는 선택자 지정 방법. class 속성 값을 사용해 선택자를 지정한다.
id 속성과는 다르게 속성 값을 중복해서 사용이 가능하다.

기본 속성 선택자

HTML 태그에서 사용하는 속성과 값을 사용해 선택자를 지정하는 방법.
속성과 함께 속성 값도 같이 지정할 수 있다.
위의 태그 선택자, 아이디 선택자, 클래스 선택자에서도 이 방식을 활용해 속성값을 함께 지정해서 사용할 수는 있다.

조합 선택자 사용하기

그룹 선택자

여러 선택자를 하나로 그룹 지을 때 사용한다.
선택자끼리의 구분은 `, `을 사용한다.

자식 선택자

부모 요소의 하위에 있는 자식 요소에 스타일 적용할 때 사용한다.

부모 선택자 안에 있는 해당 자식 선택자들을 선택해 적용할 수 있다.

하위 선택자

자식 선택자와 비슷한 개념이지만 좀 더 확장된 선택자라고 볼 수 있다.
하위 선택자는 자식 선택자보다 더 하위에 있는 선택자들을 선택할 수 있다.

인접 형제 선택자

이전 선택자 요소 바로 다음에 있는 대상 선택자 요소를 지정한다.

그 다음 요소는 선택이 되지 않는다.

일반 형제 선택자

이전 선택자 뒤에 오는 모든 형제 관계의 대상 선택자를 지정한다.

여기서는 이 태그까지 선택이 된다.

span으로 다시 적용 시켜보았다.
이전 선택자 뒤에 오는 모든 형제 관계의 대상 선택자를 지정한다.
무슨 말, 아무 말, 올해는 붉은말의 해.

가상 요소 선택자


기준 선택자를 생략할 경우 전체 선택자 `*`가 들어간 것으로 적용이 된다.
