# 실무에서 자주 사용하는 HTML 필수 태그 : 시맨틱 태그
## 시맨틱 태그
의미있게 웹 페이지를 설계하기 위해 사용하는 태그들. 의미가 담긴 태그들이다.  
대표적으로 `<table>` `<form>` `<a>` 태그들이다.  
전체적인 구조를 잡아주는 태그들도 포함된다.  
### `<header>` 태그
```html
<header>
	헤더 구성 요소
</header>
```
웹 페이지에서 헤더 영역을 구분할 때 사용한다.  
보통 웹 사이트의 최상단, 좌측에 위치한다.  
### `<nav>` 태그
```html
<nav></nav>
```
웹 페이지에서 내부의 다른 영역이나 외부 연결 링크들을 모아두는 부분.  
### `<section>` 태그
```html
<section></section>
```
논리적으로 내용 영역을 구분할 때 사용한다.  
`<section>` 태그 안에는 `<hn>`태그 중 하나를 포함해 작성한다.  
### `<article>` 태그
```html
<article></article>
```
독립적인 영역으로 구분될 때 사용된다.   
대표적인 예시로 포털사이트에서의 로그인 영역이다.  
`<section>` 태그는 웹 페이지 안에서 관련 있는 내용을 구분 한다면,  
이 태그는 어떤 웹 페이지에서든 독립적으로 사용될 수 있는 영역을 구분한다.  
### `<aside>` 태그
```html
<aside></aside>
```
주력 내용, 독립적인 내용으로 보기 어려울 때 사용한다. (article, section 태그 사용이 애매할때)  
### `<footer>` 태그
```html
<footer></footer>
```
웹 페이지에서의 푸터영역, 일반적으로 웹 페이지 최하단에 있다.  
주로 저작권 정보, 연락처, 사이트 맵 등이 담겨져 있다.  
### `<main>` 태그
```html
<main></main>
```
웹 페이지의 주요 내용을 지정할 때 사용하는 태그.  
문서에서 반복해서 등장하는 요소를 포함하면 안된다. 위에 설명한 태그들 하위에 포함될 수 없다.
