# JavaScript
웹 페이지에서 복잡한 기능을 구현할 수 있는 스크립팅 또는 프로그래밍 언어.\
웹 페이지가 단순히 정적인 정보를 표시하는 것 이상으로 콘텐츠 업데이트, 대화형 지도, 애니메이션 그래픽, 스크롤링 비디오 주크박스 등에 사용이 된다.

**HTML**\
문단, 제목, 데이터 표를 정의, 페이지에 이미지, 동영상 삽입 등 ...\
웹 콘텐츠 구성하고 의미를 부여하는 마크업 언어.\
**CSS**\
배경색, 글꼴 등 HTML 콘텐츠에 스타일을 적용하는 스타일 규칙 언어.\
**JavaScript**\
동적으로 변경되는 콘텐츠 제작, 멀티 미디어 제어, 이미지에 애니메이션 적용 등 ...\
스크립팅 언어.

## Application Programing Interface (API)
개발자가 구현하기 어렵거나 불가능한 프로그램을 구현할 수 있도록 미리 만들어서 제공.
### 브라우저 API
웹 브라우저에 내장, 주변 컴퓨터 환경의 데이터를 노출하거나 복잡한 작업을 수행.\
**DOM API (Document Object Model)**\
HTML 과 CSS 조작, 생성 및 변경, 제거를 할 수 있으며, 페이지에 새 스타일을 동적으로 적용 할 수 있다.
> 페이지에 팝업 창이 나타냄, 새로운 콘텐츠 표시 등

**Geolocation API**\
지리 정보를 가져올 수 있음.\
Google 지도에서 사용자 위치를 찾아 지도에 표시하는 방법.\
**Canvas** / **WebGL API**\
애니메이션 그래픽 제작.

...

### 서드파티 API
기본적으로 브라우저에 내장되어 있지 않으며, 웹 어딘가에서 해당 코드와 정보를 가져와야 함.
> Twitter API로 최신 트윗을 웹 사이트가 보여줄 수 있도록 구현.\
> 구글 지도 API로 웹 사이트에 지도를 삽입, 지도 관련 기능을 추가할 수 있음.
