프로그래밍 언어 (programming language) 기계에 명령 또는 연산을 수행하게 할 목적으로 설계, 컴퓨터와 의사소통 할 수 있게 한다.  
(컴퓨터랑 대화하는거 같아서 뭔가 멋있어보여 재미를 느꼈다.)  
# Javascript 기초 문법
## 변수, 상수 : 데이터 처리를 위한 데이터 저장 공간
### 변수 variant
변하는 수, 값이 변하는 데이터를 저장하고 관리하기 위한 공간.  
```js
var 변수명 = 변수값;

var num = 10 + 20; // 'num'이라는 변수이름으로 10 + 20 연산 값을 저장.
```
#### 문법 분석
- var : 키워드
- num (변수명) : 식별자
- `=` : 연산자
- 10 + 20 : 표현식
- `;` : 세미콜론

##### 키워드 keyword
자바스크립트에서 어떤 역할이나 기능이 정해진 특별한 단어. 예약어 (reserved word)라고도 불린다.  
(변수 할당할때 var 키워드를 사용함)  

##### 식별자 identifier
자바스크립트 내부에서 부여되는 변수, 함수 등의 이름.  
추상적인 메모리 공간을 사용자가 쉽게 식별할 수 있는 식별자 (이름)을 사용해 나타낸다.  
키워드에 역할에 따라 다르게 불려지게 되는데, 변수를 생성하는 `var`, `let`, `const` 등의 키워드와 함께 사용시에는 변수명이라고 불리게 된다.  
> **강제적 식별자 명명 규칙**  
> - 키워드는 식별자로 사용할 수 없다. (예 : var, let, const)  
> - 식별자 공백 포함 불가 (예: 변수 명)  
> - 식별자 첫 글자는 영문 소문자, _(언더스코어), $ 기호만 사용 가능 (사용 X 예시 : *name, Email)  

> **관용적 식별자 명명 규칙**  
> - 식별자는 영문으로만 작성 (예: 이름, 나이 -> name, age)  
> - 식별자는 의미 있는 단어로 작성 (예: a, b -> name, age)  

강제 규칙은 지키지 않을 경우 에러 발생이 되어 꼭 지켜야하지만,  
관용 규칙은 되도록 지키는게 좋다. (필수는 아님)  
식별자 표기법은 카멜 표기법, 언더스코어 표기법, 파스칼 표기법이 있다. (이건 따로 한번 정리해볼 예정)  
주로 자바스크립트에서는 파스칼 표기법, 각 단어의 첫 글자를 대문자로 작성하는 방식을 사용한다. (FirstName, LastName)  

##### 연산자 operator
어떠한 연산 작업을 할때 사용하는 기호.  
`=` 기호는 우변에 있는 데이터를 좌변에 할당하는 역할, 할당 연산자라고도 불린다.  
##### 표현식 expression
표현식을 실행해 하나의 값을 만드는 과정에서 하나의 값을 반환하는 식, 코드를 의미한다. (평가 값을 반환)  
##### 값 value
더 이상 평가할 수 없는 데이터.  
10 + 20은 30이라는 결과 값으로 평가 -> 표현식  
표현식의 결과로 도출된 30 -> 값 (더 이상 평가 할 수 없음)  
##### 세미콜론 semicolon
자바스크립트에서 하나의 문(문법)이 끝났음을 의미.  
1개의 명령 = 하나의 문법. `;` 그 끝에 기호를 붙여 하나의 명령문이 끝났음을 명시적으로 표현한다.  

### 선언, 할당, 초기화
변수를 생성, 값을 저장할때 var, let, const 키워드를 사용해 식별자 지정하는 행위는 변수를 선언하는 것이다.  
할당연산자 '='로 변수 공간에 대입 및 저장하는 것을 값을 할당한다고 말한다.  
```js
// 변수 선언
var num; // 값을 할당하지 않고 선언하는게 가능하다.

// 변수 초기화
var num = 10 + 20; // 선언과 할당을 한번에 진행.
```

## 자료형 data type : 데이터 종류
### 기본 자료형 (primitive data type)
#### 문자열 string
```js
let string1 = "Hello, world!";
let string2 = 'hello, world!';

console.log(string1);
console.log(string2);
```
텍스트 형태로 정의할때 사용. "큰따옴표", '작은따옴표' 를 사용해 string임을 나타낸다.  
큰따옴표를 사용할땐 큰따옴표로 마무리, 작은따옴표로 시작할땐 작은따옴표로 마무리 해야한다. (같은 친구들로 묶어줘야한다.)  
```js
let str1 = '문자열은 큰따옴표로(")로 묶어주거나'
let str2 = "작은따옴표(')로 묶어줘야한다."

console.log(str1)
console.log(str2)
```
문자열 안에 따옴표를 사용하게 될때에는 서로 상반된 친구들로 묶어주면 인식을 한다.  

#### 이스케이프 문자
```js
let string3 = '문자열은 큰따옴표(")나 작은따옴표(\')로 감싸면 됩니다.';

console.log(string3);
```
웹 브라우저가 사용자 의도와 다르게 문자열 해석할 때 사용한다.  
문자열 표시에서 중복으로 겹쳐질때 기호 앞에 `\` 붙여주면 중간에 끊기지 않고 문자열로 인식해 전체 출력이 된다.  
따옴표들 뿐만이 아니라 `\n` (: 줄바꿈) , `\t` (: 수평 탭) , `\\` (: 역슬래시)에도 사용이 가능하다.  

#### 탬플릿 문자열
```js
let string4 = `문자열은 큰따옴표(")나 
작은따옴표(')로 감싸면 됩니다.
enter도 적용이 됩니다.`;

console.log(string4);
```
ES6에 추가된 \` 백틱으로 문자열을 정의 하는 방법.  
큰따옴표나 작은따옴표가 있어도 영향을 받지 않으며, enter시 줄바꿈 적용이 된다.  
`${}` 문법 사용해 문자열에 변수 또는 식을 넣을 수 있다.  
```js
let x = 3;
let y = 4;
let string5 = `${x} * ${y} = ${x * y}`

console.log(string5)
```
파이썬에서의 f-string 느낌인듯 하다.  
#### 문자열 연결 연산자 + 
```js
// 문자열 연결 연산자 (+)
let str3 = '문자열' + '합치기';

console.log(str3);
```
`+` 를 사용해 문자열끼리 합칠 수 있다.  
#### 숫자형
자바스크립트에서는 정수와 실수 구분 없이 하나의 숫자형으로 취급한다.  
실수를 부동 소수점 방식으로 표현하기 때문에 실수 계산이 정확하지 않다.  
```js
let num = 0.1 + 0.2;

console.log(num);
```
#### 논리형
```js
let bool1 = true;
let bool2 = false;

console.log(bool1);
console.log(bool2);

let bool3 = 10 < 20;
let bool4 = 10 > 20;

console.log(bool3);
console.log(bool4);
```
참 또는 거짓에 해당하는 논리 값 true, false.  
두 값만 존재한다.  

논리적인 연산으로 값을 산출해낼 수 있다.  
#### undefined
```js
let empty;

console.log(empty);
```
변수나 상수에 데이터를 할당하지 않을 경우, 자바스크립트 내부에서 임시로 데이터 할당되는 값.  

#### null
```js
let empty2 = null;

console.log(empty2);
```
변수나 상수를 선언, 의도적으로 공간을 비워둘 때 사용한다.  

### 참조 자료형 (reference data type)
#### 객체 object
기본 자료형을 제외한 거의 모든 데이터와 자료구조는 객체 형태를 띄고 있다.  
자바스크립트의 핵심적인 자료형이며 '배열', '객체 리터럴', '함수'가 있다.  

##### 배열 array
```js
let studentScore = [80, 70, 90, 60]; // [요소]

console.log(studentScore[2]); // 2번 인덱스 데이터 접근

let array1 = []; // 빈 배열
let array2 = ['str', 10, true, undefined, null]; // 다양한 자료형을 넣을 수 있다.
```
하나의 자료형에 여러 개의 데이터를 정의할 수 있으며, 숫자형 외 모든 자료형을 정의할 수 있다.  
배열로 정의한 데이터를 '요소', 이 요소에 접근하기 위해서는 인덱스 index 를 사용해야한다.  

인덱스는 배열에서 각 데이터가 있는 위치를 가리키고 있는데, 1부터 시작하는 것이 아닌 0부터 시작한다.  

> [80, 70, 90, 60]  
> 
> 인덱스 index 위치  
> [0, 1, 2, 3]  

2번 출력을 하면 3번째 값인 '90'이 출력이 된다.  

##### 객체 리터럴
```js
{
    key1: value1, // -> 속성
    key2: value2
}
```
키 key 와 값 value으로 이루어진 속성으로 정의 한다.  
```js
let studentScore2 = {
    koreanScore: 80,
    englishScore: 70,
    mathScore: 90,
    scienceScore: 60
};

console.log(studentScore2.koreanScore);
console.log(studentScore2['koreanScore']);
```
속성은 key-value 한 쌍으로 이루어져있기 때문에 어떠한 값이 담겨있는지 알 수 있다.  
key를 통해 value를 찾을 수 있다.  
배열처럼 `[]`로 키값을 넣을 수도, `.` 마침표로도 값을 조회할 수 있다.  

## 연산자 operator 
연산을 처리하는 데 사용하도록 미리 정의된 기호.  
### 산술 연산자
덧셈, 뺄셈, 곱셈, 나눗셈.  
수학 연산을 수행한다.  
#### 이항 산술
> `+` : 더하기  
> `-` : 빼기  
> `*` : 곱하기  
> `/` : 나누기  
> `%` : 나누기 (나머지)  
> `**` : 거듭제곱  

이항 산술 연산자는 피연산자가 2개 필요하다.  
```js
let sum = 10 + 20;
let sub = 20 - 10;
let multi = 10 * 20;
let div = 10 / 2;
let remain = 10 % 3;
let expon = 2 ** 3;
```
#### 단항 산술 연산자 (증감 연산자)
> ++ : 1 증가  
> -- : 1 감소  

피연산자 하나로 사용할 수 있는 연산자.  
변수나 상수에 할당된 데이터로만 연산할 수 있다. (숫자로는 안된다.)  
```js
let increment = 10;
increment ++; // 11

let decrement = 10;
decrement --; // 9

// 전치 연산과 후치 연산
let num2 = 10;

console.log(num2++); // 후치 연산 : 10
console.log(++num2); // 전치 연산 : 12
```
기호들이 앞에 붙는지, 혹은 뒤에 붙는지에 따라 연산 방식이 달라진다.  
`console.log(num2++);` 연산자가 뒤에 붙을 때(후치 연산), 먼저 할당 후에 뒤에 붙은 연산 된다. (증감)  
`console.log(++num2);` 연산자가 앞에 붙을 때(전치 연산), 연산 후에 할당이 된다.  

처음 출력된 10은 후치 연산으로 할당 후에 연산이 되었기에 뒤의 전치 연산에서 11 + 1 의 값이 출력이 되었다.  
### 단항 부정 연산자
```js
let num3 = -10;
num4 = -num3;

console.log(num4); // 10
```
피연산자의 부호를 부정하는 연산을 수행.  
음수는 양수로, 양수는 음수로 변환한다.  
### 대입 연산자와 복합 대입 연산자
대입 연산자는 데이터를 대입(할당)하는 연산자 `=`.  
복합 대입 연산자는 산술 연산자와 대입 연산자를 함께 사용, 산술과 할당을 한 번에 수행한다.  
```js
x += y // x = x + y 
x -= y // x = x - y
x *= y // x = x * y
x /= y // x = x / y
x %= y // x = x % y
x **= y // x = x ** y
```
```js
let num5 = 10;

console.log(num5 *= 5); // 곱한 후에 할당.
```
### 비교 연산자
피연사자들을 비교, 논리형 값 true, false를 반환.  
```js
x == y // 두 비교 값이 같을 경우 true 반환
x === y // 두 비교 값과 자료형이 같을 경우 true 반환
x != y // 두 비교 값이 다르면 true
x !== y // 두 비교 값과 자료형이 다를 경우 true
x < y // y가 클 경우 true
x > y // x가 클 경우 true
x <= y // y가 크거나 같을 경우 true
x >= y // x가 크거나 같을 경우 true
```
#### 동등 연산자(==) 일치 연산자(===) 부등 연산자(!=) 불일치 연산자 (!==)
```js
// 동등 연산자(==)와 일치 연산자(===)
console.log(10=='10'); // true
console.log(10==='10'); // false

// 부등 연산자(!=)와 불일치 연산자(!==)
console.log(10 != '10'); // false
console.log(10 !== '10'); // true
```
동등 연산자와 부등 연산자는 타입이 달라도 값을 비교해준다.  
일치 연산자와 불일치 연산자는 값이 같아도 타입이 다를 경우 'false' 출력, 값과 데이터 타입 둘을 비교한다.  
```js
console.log(10 !== '9'); // true
console.log(10 !== 9); // true
console.log(10 !== 10); // false
```
### 논리 연산자
#### AND 연산자 (&&)
왼쪽에 있는 피연산자부터 평가, 거짓일 경우 해당 거짓 피연산자를 반환한다.  
참일 경우 마지막에 평가되는 피연산자 반환.  
```js
true && true; // 마지막 피연산자 반환

true && false && true; // 두 번째 피연산자 false 반환
```
자바스크립트 자료형 중 빈 문자열 (""), undefined, 0, null 값만 false로 평가 된다.  
```js
console.log("" && "cat"); // 
console.log('cat'&&'dog'); // dog
```
#### OR 연산자 (||)
왼쪽에 있는 피연산자부터 평가해 참으로 평가된 피연산자를 즉시 반환한다.  
거짓으로 평가된 피연산자만 있을 경우, 마지막에 평가된 피연산자가 반환 된다.  
```js
false || true || false; //true
false || false; // false

"" || false || 'cat'; // cat
```
#### NOT 연산자(!)
피연산자나 식을 평가한 논리 값의 반대로 반환한다.  
(true 값을 false로, false 값을 true로 반환)  
```js
!true; // false
!(true && false); // false -> true
!(10 < 20 && 20 < 10); // false -> true
```
#### 삼항 연산자
가장 왼쪽에 있는 피연산자 (첫 번째 피연산자)의 논리 연산자 값에 따라 남은 두 피연산자를 반환한다.  
```js
x ? y : z
```
x? (x가 참일 때) y 반환, 아닐 경우 z 반환.  
```js
let score1 = 90;
let score2 = 89;

console.log(score1 >= 90 ? 'A+' : 'B'); // A+
console.log(score2 >= 90 ? 'A+' : 'B'); // B
```
